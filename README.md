# Star Walk
<b>Link:</b> https://divyansh-007.github.io/Star-Walk/

A simple project to view Astronomical Picture Of the Day by NASA.

## Tools/Technologies Used
* HTML
* CSS
* Javascript & jQuery
* NASA (APOD) API

### Features
User can everyday visit and view the APOD released by NASA.
